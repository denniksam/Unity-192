User Interface
Елементами UI є текстові блоки, images, елементи керування (кнопки, слайдери, ...)
При додаванні будь-якого (першого) UI елемента автоматичного створюються:
- Canvas - холст, на якому розміщуються елементи.
- EventSystem - система передавання подій UI до коду
Наступні елементи можуть поміщатись на наявні холсти або створювати інші

Canvas
Холст має декілька режимів відображення
- Screen Space (Overlay) - перекриття екрану. Холст видний через будь-яку камеру,
   при її будь-якій орієнтації

- Screen Space (Camera) - прив'язка до камери (уявляти як індикатор заряду для
   звичайної камери)

- World Space - холст як GameObject - присутній у світі, видний лише коли потрапляє
   у поле зору камери, причому, як рекламний банер - при погляді "зпереду"

= Ми використовуємо режим Screen Space (Camera)
При виборі цього режиму необхідно зазначити до якої камери буде прив'язано холст
(Main Camera). Холст має масштабуватись під поле зору камери
Наступний параметр - Plane Distance - відстань від камери до площини холста.
Для того щоб інформація була видна завжди вибираємо відстань у межах (0-10)
10 - це площина гри, 0 - позиція камери. Вибрали 5
